# 🌟 The Wisdom Vault

The Wisdom Vault is a simple and elegant desktop application built using **Python** and **Tkinter**. Every click reveals a meaningful message designed to inspire, motivate, and encourage users throughout their day.

## ✨ Features

- Random inspirational and motivational messages
- Prevents consecutive repeated quotes
- Clean and modern user interface
- Copy messages to the clipboard
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- Tkinter
- Random Module

## 📂 Project Structure

```
random-quote-generator/
│── main.py
│── quotes.py
│── README.md
│── requirements.txt
```

## ▶️ How to Run

1. Clone this repository:
```
git clone https://github.com/Malleswari-panditi/Random-quote-generator.git
```

2. Open the project folder.

3. Run the application:
```
python main.py
```
